# Project Title

To display the data for the IDEAL dataset: https://www.research.ed.ac.uk/en/datasets/ideal-household-energy-dataset
Firstly on a by house basis to illustrate each individual house's energy consumption. this remains a Work in progress as I was running afoul of AWS limits

Secondly to treat all the houses as a community to see what their aggregated needs were

Thirdly to forecast the communitie's energy consumption using a variety of machine learning models: Forecasting and Regression
